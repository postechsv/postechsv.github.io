MSYNC: A Generalized Formal Design Pattern for Virtually Synchronous Multirate Cyber-Physical Systems

This web page contains resources for our paper “MSYNC: A Generalized Formal Design Pattern for Virtually Synchronous Multirate Cyber-Physical Systems” by Kyungmin Bae and Peter C. Ölveczky.

Maude Models and Analysis Commands

The file model.zip contains the complete Maude source code for the airplane control system example in the paper. The archive file contains the following Maude files to run the experiments:

  • airplane-sync-simul.maude : randomized simulations to measure information loss due to input/output cutoff numbers;
  • airplane-sync-mc.maude : model checking to verify two requirements of the synchronous model; and
  • airplane-async-reach.maude : model checking of the asynchronous model
You can run these files using Maude 3.0 (or later), available here.
