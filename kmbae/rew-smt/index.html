Executable Rewriting Modulo SMT Specification for the Case Study

The following archive file contains the complete Maude source code for two examples in the paper. There are three directories in the archive: smt for the basic rewriting modulo SMT framework, ex for the running example, and cash for the CASH case study. To run these files, an alpha version of Maude integrated with CVC4 is required.
